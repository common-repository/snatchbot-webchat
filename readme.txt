Easily integrate powerful chatbots onto your Wordpress website. Just one click to add SnatchBot widget to your page.

== Description ==

SnatchBot is a fast, convenient way to embed a customizable, real-time messaging platform on your website. Customizing an existing widget or adding a new one is quick and easy. Build a bot and add it to your Wordpress website with just one click.
This plugin is an official tool developed by SnatchBot.  SnatchBot is the powerful and secure chatbot building platform.
Please note that the plugin uses your SnatchBot account data. Learn more in [Terms](https://snatchbot.me/terms/) and [Privacy Policy](https://snatchbot.me/privacy/)
The main goal of this plugin is to allow you to easily install webchat widget onto your website.
Data accessed by the plugin:
list of your bots, names and bot logos.
embed code of your bots. 
All data processing falls under Privacy policy conditions.
